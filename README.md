### `gui_server`

*   **Назначение:** Сервер, работающий на управляющем компьютере (например, вашем ПК).
*   **Функция:** Предоставляет **графический интерфейс (GUI)** для управления роботом. Пользователь управляет роботом с помощью **слайдеров** в интерфейсе.
*   **Важно:** Не поддерживает подключение к **джойстику манипулятора**. Управление осуществляется исключительно через элементы GUI (ползунки, кнопки и т.д.).

### `client`

*   **Назначение:** Клиент, запускаемый на Raspberry Pi, установленном на роботе.
*   **Функция:** Подключается к *белому* IP-адресу `RobotX`. Принимает команды управления от `gui_server`.
*   **Подключение к есп:** Нужно осуществить через **демон-процесс**, который запускается в начале `main()` функции клиента.
*   **Обработка данных:** С данными нужно работать в функции `process_control_data()`. Внутри этой функции должны находиться вызовы команд для **управления роботом по Serial** (например, отправка команд на микроконтроллер, управляющий моторами и другими исполнительными механизмами).

### `server`

*   **Назначение:** Сервер, работающий на управляющем компьютере (например, вашем ПК).
*   **Функция:** Предоставляет программу которая подключается к джойстику-манипулятору на "COM5", читает с него данные и отправляет клиенту.
